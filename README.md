## 📖 About

**Pixel Purge Play** is a fast-paced, pixel-art game that challenges your reflexes and strategic thinking. Navigate through a vibrant grid of enemies, collect power-ups, and purge your way to victory. Inspired by classic arcade games with a modern twist — every pixel counts!

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🕹️ **Retro Gameplay** | Classic arcade mechanics with smooth modern controls |
| 🎨 **Pixel Art Graphics** | Hand-crafted sprites and animations |
| 🔊 **Chiptune Soundtrack** | Original 8-bit inspired music and SFX |
| 🏆 **Leaderboards** | Compete for the high score |
| 🌈 **Multiple Levels** | Increasing difficulty and unique environments |
| ⚡ **Power-ups** | Special abilities to turn the tide |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/rishavbuilder/pixel-purge-play.git

# Navigate to the project
cd pixel-purge-play

# Install dependencies
pip install -r requirements.txt

# Run the game
python main.py
```

### Prerequisites

- Python 3.8+
- Pygame 2.0+
